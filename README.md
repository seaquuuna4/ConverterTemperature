# Конвертер температур

## Описание проекта

Графическое приложение для конвертации температуры из градусов Цельсия в градусы Фаренгейта и Кельвины.

Программа разработана на Python с использованием PyQt5. В приложении реализованы сохранение истории конвертаций в SQLite, экспорт данных в CSV и загрузка изображения через Pillow.

## Запуск проекта

1. Создайте виртуальное окружение:

`python -m venv venv`

2. Активируйте виртуальное окружение:

* Windows: `venv\Scripts\activate`
* macOS/Linux: `source venv/bin/activate`

3. Установите зависимости:

`pip install -r requirements.txt`

4. Запустите программу:

`python main.py`

## Структура проекта

* `main.py` → Точка входа, настройка QApplication, логика работы приложения и обработка событий.
* `des.py` → Интерфейс PyQt5, созданный в Qt Designer.
* `database.py` → Работа с базой данных SQLite (CRUD).
* `untitled.ui` → Файл интерфейса Qt Designer.
* `thermometer.png` → Изображение термометра.
* `requirements.txt` → Зависимости проекта.

## Возможности программы

* Конвертация °C в °F и K.
* Сохранение истории операций.
* Очистка истории.
* Экспорт истории в CSV.
* Отображение температурной шкалы.
* Работа с горячими клавишами.

## Используемые технологии

* Python
* PyQt5
* SQLite
* Pillow
